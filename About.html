More about the project:

This website was developed as part of a University project by Matthew Liberson. We were tasked with developing applications using Phillips Hue Lights and a website to host everything. HTML, CSS, Javascript, JQuery, Boostrap and Ajax were all used. Ajax was used for sending packets to the router linked up with the Phillips Hue Lights.
